== Description ==

Easily disable comments on your website

== Installation ==

1. Upload the `easy-disable-comments` folder to the `/wp-content/plugins/` directory or install directly through the plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress or by using the link provided by the plugin installer

## How to use

1. First Activate Plugin.
2. Then Click on "Settings->Easy Disable Comments" menu. Then update plugin configuration as per your need.
